Voice Assistant using Python
📌 Project Description

This project is a Custom Voice Assistant developed using Python that can recognize voice commands and perform tasks automatically.

The assistant allows users to interact with the computer using speech, making operations faster, efficient, and hands-free.

The system listens to user commands, processes them, and provides voice responses while performing requested tasks such as opening websites, searching information, telling time, and more.

Working Process:-

        The assistant listens to user input through a microphone.

        Speech is converted into text using speech recognition.

        The program processes the command.

        The assistant performs the requested task.

        The assistant responds using voice output.
